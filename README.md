## Описание
Проект состоит из 5 консольных игр:
* Проверка на четность;
* Калькулятор;
* Наибольший общий делитель;
* Арифметическая прогрессия;
* Простое ли число?.

## Минимальные требования

Node.js версии 13.2.0 и выше

## Инструкция по установке
```
make install
```
```
npm link
```
